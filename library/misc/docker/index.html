Setting up WSL2, Docker, and running your first container.

The Union toolchain is a pre-built toolchain by Shauninman. It contains everything you need to compile for the Miyoo Mini(&Plus)

WSL and Docker

What we'll be doing:

  1. Enable WSL and set WSL2 as default.
  2. Install a Linux distribution from Microsoft Store.
  3. Install Docker Desktop and enable WSL2 integration.
  4. Install Git and Make in your WSL terminal.
  5. Clone a repository and build the Docker image.
Step 1: Enable WSL and Install WSL2

  1. Enable WSL: Open PowerShell as Administrator and run: 

    wsl --install

  2. Set WSL2 as the Default Version: 

    wsl --set-default-version 2

  3. Restart Your Computer: Restart your computer to apply changes.

Step 2: Install a Linux Distribution

  1. Install a Linux Distribution: Open Microsoft Store, search for your preferred Linux distribution (e.g., Ubuntu), and install it.

  2. Set Up Your Linux Distribution: Launch the installed distribution from the Start menu and complete the initial setup (create a user account and set a password).

Step 3: Install Docker Desktop

  1. Download Docker Desktop: Download Docker Desktop from Docker's official website and install it.

  2. Enable WSL2 Integration: During the installation, ensure the option "Use the WSL 2 based engine" is checked.

  3. Finish Installation: Complete the installation and start Docker Desktop.

Step 4: Configure Docker to Use WSL2

  1. Open Docker Desktop: Go to Settings.

  2. Enable WSL2 Integration:

  3. Navigate to Resources -> WSL Integration.
  4. Ensure that the Linux distribution you installed is checked to enable integration.
Step 5: Install Make and Git in WSL

  1. Open WSL Terminal: Open your installed Linux distribution from the Start menu.

  2. Update Package Lists: 

    sudo apt update

  3. Install Git and Make: 

    sudo apt install git make

Step 6: Clone the Repository and Build the Docker Image

  1. Clone the Repository: 

    git clone https://github.com/shauninman/union-miyoomini-toolchain.git

  2. Navigate to the Directory: 

    cd union-miyoomini-toolchain

  3. Build and Run the Docker Container: 

    make shell

You're now in a containerised shell and can begin compiling; why not try this script out to compile your first Miyoo compatible binary?
